CSS Selectors

Applying CSS to Different Parts of HTML

1. Element selector: targets HTML tags (e.g., p, h1)

  1. Class selector: targets elements with a specific class (e.g., .menu)
  2. ID selector: targets an element with a specific ID (e.g., #header)
  3. Universal selector: targets all elements (*)
  4. Attribute selector: targets elements based on attributes (e.g., input[type="text"])
